## Search high quality sources
### How to find?
- Use keywords to search
- What is a good source?
    - recent
    - impact factor
    - amount of citations
    - journal
    - peer reviews
### How to read/use academic papers?
- Basic flow
    1. Read title and abstract
        - Interesting?
        - Relevant?
    2. Check figures and read captions
        - Interesting?
        - Relevant?
    3. Read introduction and conclusion
    4. Read it completely (if necessary)
- Use references to find other papers that cite this paper!
    - previous theses
    - Sites of professional, official know sources
    - Books
    - Syllabi
### Research methodology: SMART
- **Specific**
    - Clear, focused and well-defined.
- **Measurable**
    - Measurable elements, quantifiable assets that you can check
- **Acceptable**
    - Realistic, attainable for researchers, respondents, institutions.
- **Realistic**
    - Doable in the available time, resources and knowledge.
- **Time-bound**
    - Clearly time-bound so that the research can be finished or if changes happen afterwards.
