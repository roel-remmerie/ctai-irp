# Lifelong Learning

## Grading rubric
- [ ] Answers questions in the community about the research topic
- [ ] Creates a video or tutorial about a specific technology from the research to share knowledge
- [ ] Consults community sources (forums, StackOverflow, social media, ...)
- [ ] Critically evaluates community sources
- [ ] Uses e-learning platforms (e.g., Pluralsight)
- [ ] Can provide a set of community leaders in the domain
